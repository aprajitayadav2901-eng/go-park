# Go Park 🚗  
### Smart Parking Management System using Computer Vision & Machine Learning

Go Park is an intelligent parking management system designed to automate vehicle parking operations using Computer Vision, Machine Learning, and real-time parking slot monitoring. The project aims to reduce manual parking management efforts and improve parking efficiency through automation.

---

## 📌 Project Overview

This system helps in:

- Detecting vehicle entry and exit
- Monitoring parking slot occupancy
- Managing parking availability in real-time
- Reducing traffic congestion in parking areas
- Improving parking management efficiency

The project combines Python, OpenCV, and Machine Learning techniques to build a smart and scalable parking solution.

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/aprajitayadav2901-eng/go-park.git
```

Move into the project directory:

```bash
cd go-park
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
